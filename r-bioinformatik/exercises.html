Aufgabe 5


RNA-Polymerase

Aufgabe 5

Entwickeln Sie eine Funktion rnaP(x), das zu einer gegebenen DNA-Sequenz x die komplementäre mRNA-Sequenz erstellt.

Lösung 5

rnaP <- function(dna) {
  complement <- c("U", "G", "C", "A")
  names(complement) <- c("A", "C", "G", "T")

  mrna <- c()
  for (i in 1:length(dna)) {
    mrna[i] <- complement[dna[i]]
  }

  return(mrna)
}

Aufgabe 6


DNA-Triplets

Aufgabe 6

Für eine DNA-Sequenz soll eine Kennzahl erstellt werden: Die DNA wird hierfür in Triplets eingeteilt. Falls das Triplet einzig paarweise verschiedene Basen besitzt, wird die Kennzahl um eins erhöht.
Implementieren Sie die Funktion kennzahl(x), die die so definierte Kennzahl für eine gegebene DNA-Sequenz bestimmt.

Lösung 6 (I)

kennzahl <- function(dna) {
  wert <- 0
  for (i in seq(1, length(dna), 3)) {
    if (dna[i] != dna[i+1] && 
        dna[i] != dna[i+2] && 
        dna[i+1] != dna[i+2]) {
      wert <- wert + 1
    }
  }
  return(wert)
}

Lösung 6 (II)

kennzahl <- function(dna) {
  m <- matrix(dna, 3,)
  ungleichheit <- function(v) {
    return(v[1] != v[2] && v[1] != v[3] && v[2] != v[3])
  }
  bool <- apply(m, 2, ungleichheit)
  return(sum(bool))
}

nextFib <- function(vektor) {
  l <- length(vektor)
  return(vektor[l-1]+vektor[l])
}

fibonacciVektor <- function(x) {
  vektor <- c(0,1)
  neueZahl <- nextFib(vektor)
  while (neueZahl < x) {
    vektor <- append(vektor, neueZahl)
    neueZahl <- nextFib(vektor)
  }
  return(vektor)  
}

Aufgabe 10


Dotplot

Aufgabe 10 (I)

Schreiben Sie eine Funktion, die für Eingabe zweier Nukleotid-Sequenzen als String einen Dotplot erzeugt.

Lösung 10 (I)

dotplot <- function(s1, s2) {
  s1c <- unlist(strsplit(s1, ""))
  s2c <- unlist(strsplit(s2, ""))
  
  x <- c()
  y <- c()
  for (i in 1:length(s1c)) {
    for (k in 1:length(s2c)) {
      if (s1c[i] == s2c[k]) {
        x <- c(x, i)
        y <- c(y, k)
      }
    }
  }

  plot(x, y)
}

Aufgabe 12


Globales Alignment

Aufgabe 12 (I)

Was bedeutet Alignment bzw. Alinieren?

Lösung 12 (I)

"Alignment bzw. Alinieren bedeutet das gegeneinander Ausrichten von zwei oder mehr Sequenzen, so dass eine große Übereinstimmung zwischen den Sequenzen gefunden wird."

Aufgabe 12 (II)

Geben Sie die Rekursionsgleichung zur berechnung eines optimalen globalen Alignments zweier Sequenzen bzgl. der Einheitskostenfunktion an.

Lösung 12 (II)

Basisfälle
E(0, 0) = 0
E(i, 0) = i
E(0, j) = j
Rekursion
top = E(i, j-1) + 1
left = E(i-1, j) + 1
topLeft = E(i-1, j-1) + K(s1[i], s2[j])
E(i, j) = min(top, left, topLeft)

Aufgabe 12 (III)

Was bedeutet Editierdistanz?

Lösung 12 (III)

Die Editierdistanz ist die Anzahl der Mismatches, Deletionen und Insertionen, die für ein Alignment nötig sind. (Dies entspricht der Anzahl der Änderungen, um eine Sequenz in die andere zu überführen.)
